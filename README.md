# V13-Discord-Bot
A Template Discord Bot Using Latest Version Of Discord.js (Slash Commands)

## Setup 
Edit The `config.json` And You're Ready To Go
```
{ 
"token": "",
"owners": [""],
"mongo": "",
"readyChannel": "",
"bugChannel": "",
"joinChannel": "",
"leaveChannel": "",
}
```
Install All The Packages By Doing `npm install`
Start The Bot By `node index.js`
